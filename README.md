# 🔴 PokéNext: Interactive Pokedex

PokéNext is a high-performance, visual encyclopedia for Pokémon enthusiasts. It leverages real-time data fetching to provide a seamless experience for browsing, searching, and analyzing Pokémon species and their attributes.

## 🚀 Live Demo

Check out the live application here: **[View Live Project](https://pokedex-liard-seven-30.vercel.app/)**

## ✨ Features

* **Dynamic Data Fetching:** Integrated with the **PokéAPI** to provide up-to-date stats, abilities, and sprites.
* **Real-time Search:** Instantly filter through hundreds of Pokémon using a high-speed search bar.
* **Detailed Stat Visualization:** Interactive displays for HP, Attack, Defense, and more.
* **Responsive Grid Layout:** A mobile-first design that looks great on any screen size.
* **Efficient Caching:** Optimized performance to reduce API load and improve navigation speed.

## 🛠️ Tech Stack

* **Frontend:** React.js
* **Build Tool:** Vite (for lightning-fast HMR)
* **API:** PokéAPI (RESTful)
* **State Management:** React Hooks (`useState`, `useEffect`)
* **Styling:** CSS3 / Modern Flexbox & Grid
